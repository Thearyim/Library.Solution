## Description
This is an application to catalog a library's books and let patrons check them out.
A user can create, read, update, delete and list books in the catalog to keep track of inventory.
A user can search for a book by author or title.
A user can enter multiple authors for a book.
A user can check out a book.
A user can find out how many copies of a book are on the shelf.
A user can see a history of all the books checked out.
A user can find out the due date for books that are checked out.
A user can see a lit of overdue books.
